# deno-web-vite-react-swc-express-typescript-template

## Description
A react template without typecheck for the vite packager.

Feature
- swc skip typechecking

## Tech stack
- deno
- express
- react
- typescript
- vite

## Docker stack
- archlinux

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
